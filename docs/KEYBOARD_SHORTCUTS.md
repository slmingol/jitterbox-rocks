# Keyboard Shortcuts

The Music Trivia Game now supports full keyboard navigation during gameplay for a faster, more accessible experience.

## During Gameplay

### Multiple Choice Questions

| Key | Action |
|-----|--------|
| **1-9** | Select option by number (e.g., press `1` for first option, `2` for second, etc.) |
| **↑ / ↓** | Navigate up/down through options |
| **Enter** | Submit your selected answer |

### Text Input Questions

| Key | Action |
|-----|--------|
| **Enter** | Submit your typed answer |

### Audio Questions ("Name That Tune")

| Key | Action |
|-----|--------|
| **Enter** | Submit your typed answer |

### After Answering

| Key | Action |
|-----|--------|
| **Enter** | Continue to next question or finish game |

## Visual Hints

When playing a game, you'll see keyboard shortcut hints displayed on the screen:

- **Multiple Choice**: Shows "⌨️ **Keyboard shortcuts:** Press 1-4 to select, ↑↓ arrows to navigate, Enter to submit"
- **Text/Audio Input**: Shows "⌨️ **Keyboard shortcut:** Press Enter to submit your answer"
- **After Answering**: Shows "Press **Enter** to continue"

## Benefits

- ⚡ **Faster gameplay** - No need to reach for the mouse
- ♿ **Accessibility** - Better support for keyboard-only users
- 🎯 **Focus** - Keep your hands on the keyboard while playing
- 🎮 **Game-like experience** - Navigate like a traditional game

## Tips

1. **Speed players**: Use number keys for multiple choice questions to answer quickly
2. **Navigation**: Arrow keys let you review options before selecting
3. **Text input**: Type your answer and press Enter - no need to click "Submit"
4. **Flow state**: Once you're in a game, you can complete it entirely with the keyboard

Enjoy the enhanced gameplay experience! 🎵🎮
